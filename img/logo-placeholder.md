# Logo Placeholder

Este archivo sirve como placeholder para el logo de Figger Energy SAS.

## Instrucciones para el Logo

El logo debe cumplir con las siguientes especificaciones:

- **Formato**: PNG con fondo transparente
- **Dimensiones**: 200x60 píxeles (proporción 10:3)
- **Estilo**: Institucional y profesional
- **Colores sugeridos**: 
  - Verde institucional (#2c5f2d)
  - Verde secundario (#4a8b3a)
  - Verde claro (#97bf47)

## Elementos a Incluir

1. **Texto**: "Figger Energy SAS"
2. **Subtítulo**: "Gobierno de Colombia" (opcional)
3. **Símbolo**: Elemento gráfico relacionado con:
   - Energía renovable
   - Medio ambiente
   - Monitoreo/vigilancia
   - Colombia

## Ubicación del Archivo

Guarde el logo como: `img/logo.png`

Si no se proporciona un logo, el sitio web funcionará correctamente mostrando solo el texto institucional.
